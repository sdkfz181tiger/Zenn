# Python3で複数のデータを使ってみよう_前編

今回のテーマは"複数のデータ"です。
一度に沢山のデータをまとめて管理する事ができるようになります。

## リスト

リストを使う事で、複数のデータをまとめて管理することができます。
"["と、"]"の間に、","区切りで値を並べて記述します。

```python:main.py
member = ["まる子", "たま", "丸尾"]
print(member) # 全てのデータを確認する
# ['まる子', 'たま', '丸尾']
```

リストにある値の1つだけにアクセスする方法は次のとおりです。
先頭の要素は[0]、2番目の要素は[1]、3番目の要素は[2]という具合にアクセスします。

```python:main.py
member = ["まる子", "たま", "丸尾"]
print(member[0]) # まる子
print(member[1]) # たま
print(member[2]) # 丸尾
```

リストの値は変更可能(mutable)なので、
好きなタイミングでその要素を入れ替える事が可能です。

```python:main.py
member = ["まる子", "たま", "丸尾"]
member[1] = "ハマジ"
member[2] = "ブー太郎"
print(member)
# ['まる子', 'ハマジ', 'ブー太郎']
```

### リストに用意された様々なメソッド

リストには、要素の追加や削除等、様々な操作を可能にする複数の"メソッド"が存在します。
ここでは、主要なメソッドに絞って紹介します。

#### append(要素を追加)

リストの末尾にデータを追加する場合は、"append()"を使います。

```python:main.py
member = ["まる子", "たま", "丸尾"]
member.append("花輪")
print(member)
# ['まる子', 'たま', '丸尾', '花輪']
```

#### insert(要素の挿入)

リストの好きな位置にデータを追加する場合は、"insert()"を使います。

```python:main.py
member = ["まる子", "たま", "丸尾", "花輪"]
member.insert(3, "みぎわ")
print(member)
# ['まる子', 'たま', '丸尾', 'みぎわ', '花輪']
```

#### extend(iterableの全データを追加する)

リストに別のリスト等を追加する場合は、"extend()"を使います。

```python:main.py
member = ["まる子", "たま", "丸尾"]
others = ["永沢", "藤井", "山根"]
member.extend(others)
print(member)
# ['まる子', 'たま', '丸尾', '永沢', '藤井', '山根']
```

#### remove(要素の削除)

リストに含まれる特定の要素を削除する時は、"remove()"を使います。

```python:main.py
member = ["まる子", "たま", "丸尾"]
member.remove("たま")
print(member)
# ['まる子', '丸尾']
```

#### pop(指定位置の要素を取り出す)

リストから指定位置にある要素を取り出し、その要素を返します。
取り出したデータはリストから削除されます。
インデックスの指定が無い場合は、最後尾の要素を返します。

```python:main.py
member = ["まる子", "たま", "丸尾"]
print(member.pop(1)) # たま
print(member.pop()) # 丸尾(インデックスの指定が無い場合は最後尾)
print(member) # ['まる子']
```

#### clear(全てを削除)

リストをクリアします。

```python:main.py
member = ["まる子", "たま", "丸尾"]
member.clear()
print(member)# []
```

#### index(存在する最初のインデックスを返す)

リストに存在する最初のインデックスを返します。

```python:main.py
member = ["まる子", "野口", "たま", "野口", "丸尾", "野口"]
print(member.index("野口"))# 1
```

#### count(特定の値の総数を返す)

リストに存在する要素の総数を返します。

```python:main.py
member = ["まる子", "野口", "たま", "野口", "丸尾", "野口"]
print(member.count("野口"))# 3
```

#### sort(要素を並び替える)

リストを文字コード順(辞書順)に並び替えます。

```python:main.py
member = ["まる子", "たま", "丸尾"]
member.sort()
print(member)
# ['たま', 'まる子', '丸尾']
```

#### reverse(要素を逆順にする)

リストを単純に逆順にして並び替えます。

```python:main.py
member = ["まる子", "たま", "丸尾"]
member.reverse()
print(member)
# ['丸尾', 'たま', 'まる子']
```

## スライス

スライスを使う事で、リスト(タプル)の一部分を取り出すことができます。
":"を使って、取り出す開始位置と終了位置を指定するだけです。
開始位置から、終了位置の1つ手前までがその対象になります。

```python:main.py
リスト[開始:終了] # 開始位置 ~ 終了位置の1つ手前まで
```

使用例は次の通りです。

```python:main.py
member = ["まる子", "たま", "丸尾", "大野", "杉山"]
print(member[1:3]) # ['たま', '丸尾']
print(member[3:])  # ['大野', '杉山']
print(member[:2])  # ['まる子', 'たま']
print(member[-2:]) # ['大野', '杉山']
```
