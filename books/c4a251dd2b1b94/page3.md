# Python3で挨拶してみよう

今回からコーディングのスタートです。
(お待たせしました!!)

## プログラムを格納するフォルダを作る

先ずははじめの一歩です。
あなたのPCのデスクトップに、
次の手順で空のフォルダを作ります。

Macの場合: Desktop -> 右クリック -> "新規フォルダ"
Winの場合: Desktop -> 右クリック -> "新規フォルダ"

新しいフォルダの名前は何でも良いのですが、
ここではわかりやすく"hello_python3"としておきます。

## プログラムを記述するファイルを作る

次に、"hello_python3"フォルダの中にテキストファイルを作ります。
新しいファイルの名前は、"main.py"にしておきます。
今後はこのファイルにコードを記述していきます。

## プログラムを書いてみる

"main.py"を、何かしらのテキストエディタ等で開き、
次のコードを入力します。
(コピー&ペーストでも良いですよ)

```python: main.py
print("Hello, Python3!!")
```

記述されているコードが今回の主役、"print(...)"という命令です。
最も簡単な命令の一つで「文字を表示するコード」です。
"..."の部分に、表示したい文字を記述します。

## プログラムを実行してみる

"main.py"を保存(忘れずに!!)したら、
ターミナル/コマンドプロンプトを開き、次のコマンド(2つあります)を実行します。

"hello_python3"フォルダに移動するコマンドと、
Python3を実行するコマンドを実行します。

```command:コマンド(Macの場合)
# フォルダに移動する
$ cd ~/Desktop/hello_python3
# Python3を実行する
$ python3 main.py
# Hello, Python3!! <- 実行結果
```

```command:コマンド(Winの場合)
# フォルダに移動する
C:\> cd C:\Users\<あなたのユーザー名>\Desktop\hello_python3
# Python3を実行する
C:\> python main.py
# Hello, Python3!! <- 実行結果
```

"Hello, Python3!!"の文字が表示されていれば大成功です!!
最初の難関をクリアした事になります!!(おめでとうございます!!)

## エラーが出たら...

ここで、よくあるエラーとして次のものがあげられます。

"No such file or directory"
フォルダ移動できてない可能性があります。
フォルダ名を再確認し、もう一度 cd コマンドで移動しましょう。

"python: command not found"
PATHが未設定の可能性があります。
第2章を参考に、Python3を再インストールしましょう。
