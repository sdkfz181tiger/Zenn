# Pythonでクラスを使ってみよう

今回は、クラスについて学びます。
クラスを学習するには、"オブジェクト"という概念を理解する必要があります。

## オブジェクトとは

"オブジェクト"とは、データと、そのデータを扱う関数をひとまとめにしたものです。
ゲームのキャラクター1体や、アプリの1ユーザーをイメージすると分かりやすいでしょう。

## クラスとは

"クラス"とは、先述したオブジェクトの"設計図"のことです。
設計図(クラス)を元にして、実際のオブジェクト(インスタンス)を作成して使います。

クラスを適切に使うことで、データの保護(カプセル化)や機能の分割など、
より扱いやすいプログラムを書けるようになります。

### 1, クラスの定義

クラスは、"class"キーワードを使って次の様に定義します。

```python:main.py
class クラス名:
    """ クラスの定義 """
```

### 2, 初期化メソッドとは

インスタンス化時、一度だけ実行される特別な関数を"初期化メソッド"と呼びます。
初期化メソッドでは、その名の通り、データの初期化等を行います。

初期化メソッドは、"def"キーワードの後に、"\_\_init\_\_"という固有のメソッド名をつけます。
第一引数には、"self"を忘れずに定義します。
"self"には、そのメソッドを呼び出した"インスタンス自身"が入ります。

```python:main.py
class クラス名:
    """ クラスの定義 """
    def __init__(self):
        """ 初期化メソッド """
```

### 3, インスタンス変数とは

クラスに定義したデータのことを、"インスタンス変数"と呼びます。

"インスタンス変数"は、初期化メソッドの部分で、
"self.変数名"のようにして宣言、初期化します。

```python:main.py
class クラス名:
    """ クラスの定義 """
    def __init__(self):
        """ 初期化メソッド """
        self.変数名 = "データ"
```

### 4, インスタンスメソッドとは

クラスに定義した関数のことを、"インスタンスメソッド"と呼びます。

インスタンスメソッドは、"def"キーワードを使って次のように定義します。
初期化メソッドと同様に、第一引数には、"self"を忘れずに定義します。

```python:main.py
class クラス名:
    """ クラスの定義 """
    def __init__(self):
        """ 初期化メソッド """
        self.変数名 = "データ"

    def インスタンスメソッド名(self):
        """ インスタンスメソッド """
```

## クラスを使ってみる

次の例は、クラスの定義からインスタンス化、そして実行までの例です。

```python:main.py
class User:
    """ クラスの定義 """
    def __init__(self, firstname, nickname):
        """ 初期化処理 """
        self.firstname = firstname# インスタンス変数
        self.nickname = nickname# インスタンス変数

    def greeting(self):
        """ インスタンスメソッド """
        print(self.firstname, self.nickname)
        print(f"私の苗字は、{self.firstname}で、ニックネームは、{self.nickname}だよ!!")

# クラスの使い方

user1 = User("桜", "まる子") # インスタンス化
user1.greeting() # インスタンスメソッドの実行
# 私の苗字は、桜で、ニックネームは、まる子だよ!!

user2 = User("穂波", "たまちゃん")
user2.greeting()
# 私の苗字は、穂波で、ニックネームは、たまちゃんだよ!!

user3 = User("丸尾", "丸尾くん")
user3.greeting()
# 私の苗字は、丸尾で、ニックネームは、丸尾くんだよ!!
```
