# Pythonでクラスを使ってみよう

今回は、クラスについて学びます。
クラスを学習するには、"オブジェクト"という概念を理解する必要があります。

## オブジェクトとは

"オブジェクト"とは、データと関数を一つにまとめたものです。
一つにまとめることで、データ自体の保護(カプセル化)や、機能の分担等が容易になります。

## クラスとは

"クラス"とは、先述したオブジェクトの"設計図"のことです。
この"設計図"を元にして、オブジェクトを生成(以下インスタンス化)して利用します。

### 1, クラスの定義

クラスは、"class"キーワードを使って次の様に定義します。

```python:main.py
class クラス名:
    """ クラスの定義 """
```

### 2, 初期化メソッドとは

インスタンス化時、一度だけ実行される特別な関数を"初期化メソッド"と呼びます。
初期化メソッドでは、その名の通り、データの初期化等を行います。

初期化メソッドは、"def"キーワードの後に、"\_\_init\_\_"という固有のメソッド名をつけます。
第一引数には、"self"を忘れずに定義します。
"self"には、"そのメソッドを呼び出したインスタンス自身"が入ります。

```python:main.py
class クラス名:
    """ クラスの定義 """
    def __init__(self):
    	""" 初期化メソッド """
```

### 3, インスタンス変数とは

クラスに定義したデータのことを、"インスタンス変数"と呼びます。

"インスタンス変数"は、初期化メソッドの部分で、
"self.変数名"のようにして宣言、初期化します。

```python:main.py
class クラス名:
    """ クラスの定義 """
    def __init__(self):
        """ 初期化メソッド """
        self.変数名 = "データ"
```

### 4, インスタンスメソッドとは

クラスに定義した関数のことを、"インスタンスメソッド"と呼びます。

インスタンスメソッドは、"def"キーワードを使って次のように定義します。
初期化メソッドと同様に、第一引数には、"self"を忘れずに定義します。

```python:main.py
class クラス名:
    """ クラスの定義 """
    def __init__(self):
        """ 初期化メソッド """
        self.変数名 = "データ"

    def インスタンスメソッド名(self):
        """ インスタンスメソッド """
```
