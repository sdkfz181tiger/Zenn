# Python3で条件分岐を使ってみよう

今回のテーマは"条件分岐"です。
条件分岐を使うことで、複雑な処理を実装することができます。

## 真偽値と比較演算子

条件分岐の話をする前に、"真偽値"と"比較演算子"について解説をします。

### 真偽値とは

真偽値とは、"真"か"偽"のいずれか2種類のデータだけを表すデータ型のことです。

"真"は真実(しんじつ)の意味を表す"True"(トゥルー)で表現されます。
反対に、"偽"は偽物(にせもの)の意味を表す"False"(フォルス)で表現されます。

マルバツクイズの、マル(あたり)とバツ(はずれ)の関係と考えるとイメージしやすいです。
条件分岐では、この真偽値を使って判定を行います。

### 比較演算子とは

比較演算子とは、その名の通りデータ同士を"比較"する記号のことです。
比較した結果は、"True"または"False"の真偽値として返されます。
条件分岐では、この判定結果に従い処理の実行可否を決定します。

主な比較演算子は次のとおりです。

| 比較演算子 | 記述例 | 意味 |
| ---- | ---- | ---- |
| == | a == b | a, bの値が同じであればTrue, それ以外はFalse |
| != | a != b | a, bの値が違う場合はTrue, それ以外はFalse |
| < | a < b | aがb未満であればTrue, それ以外はFalse |
| <= | a <= b | aがb以下であればTrue, それ以外はFalse |
| > | a > b | aがbより大きければTrue, それ以外はFalse |
| >= | a >= b | aがb以上であればTrue, それ以外はFalse |

## 条件分岐を使ってみる

条件分岐の具体的な記述方法について見ていきます。

### if文を使ってみる

条件分岐では、"if"キーワードを使って次のように記述します。
インデント(字下げ)によって、「条件が"True"のときに実行される処理のかたまり」を表します。

```python:main.py
if 条件:
    条件が"True"だった時に実行される
```

比較演算子を使った簡単な例をみてみます。

次の例では、条件が"True"と判定され、print()文が実行されます。

```python:main.py
age = 18
if age >= 18:
    print("あなたは成人です!!")
# 実行結果: あなたは成人です!!
```

しかし、ageの値を17にして実行した場合、
条件が"False"となり、print()文は実行されません。

```python:main.py
age = 17
if age >= 18:
    print("あなたは成人です!!")
# 実行結果: 条件が"False"のため、何も表示されません
```

### else文を使ってみる

"else"を使うと、より柔軟な条件分岐を作ることができます。
"else"は、"~じゃないとき"という意味です。

```python:main.py
age = 18
if age >= 18:
    print("あなたは成人です!!")
else:
    print("あなたは未成年です!!")
# 実行結果: あなたは成人です!!
```

次の例では、先ほどの処理と似ていますが、
条件が"False"のときに実行される処理も書けるようになっています。
ageの値を17に変更して実行するとその動きが理解できます。

```python:main.py
age = 17
if age >= 18:
    print("あなたは成人です!!")
else:
    print("あなたは未成年です!!")
# 実行結果: 条件が"False"のため、"あなたは未成年です!!"が表示される
```

こちらの場合は、条件式が"False"となる為、
"あなたは未成年です!!"が表示されます。

"if-else"を使うことで2択で処理を分岐させることが可能になるので覚えておきましょう。

### elif文を使ってみる

"elif"は、"if"の条件が"False"のときに、続けて判定したい条件を書くためのものです。
"if"が"False" -> 次に"elif"の条件を判定 -> "True"ならその処理が実行されます。

```python:main.py
age = 17
if age >= 18:
    print("あなたは成人です!!")
elif age >= 15:
    print("あなたは少年です!!")
else:
    print("あなたは児童です!!")
# 実行結果: 最初の判定が"False"、次の判定が"True"となる為、
#         "あなたは少年です!!"が表示される
```

最後の"else"文は、全ての判定が"False"であった場合に実行されることになります。

```python:main.py
age = 14
if age >= 18:
    print("あなたは成人です!!")
elif age >= 15:
    print("あなたは少年です!!")
else:
    print("あなたは児童です!!")
# 実行結果: 全ての判定が"False"となる為、
#         "あなたは児童です!!"が表示される
```

"elif"文は特別に2つ以上を好きなだけ続けて記述することができます。
判定する順番が重要になるので、気をつけて実装をしましょう。

```python:main.py
age = 14
if age >= 18:
    print("あなたは成人です!!")
elif age >= 15:
    print("あなたは少年です!!")
elif age >= 6:
    print("あなたは児童です!!")
else:
    print("あなたは幼児です!!")
# 実行結果: 3つ目の判定で"True"となる為、
#         "あなたは児童です!!"が表示される
```

"if-elif-else"を理解すれば、複雑な分岐処理を実装できるようになります。

## 論理演算子

論理演算子を使うことで、
複数の条件を組み合わせて判定をすることができます。

主な論理演算子は次のとおりです。

| 論理演算子 | 記述例 | 意味 |
| ---- | ---- | ---- |
| not | not a | aの判定が逆転します。("True"であれば"False") |
| or | a or b | aもしくはbの判定のどちらかが"True"であれば"True" |
| and | a and b | aとbの判定の両方が"True"であれば"True" |

### notを使ってみる

"not"を使った記述例は次の通りです。

```python:main.py
age = 18
if not age >= 18:
    print("あなたは未成年です!!")
# 実行結果: ageの値が、18以上"でない"ときに、
#         "あなたは未成年です!!"が表示される
```

### orを使ってみる

"or"を使った記述例は次の通りです。

```python:main.py
age = 25
if age < 18 or age >= 65:
    print("あなたは未成年かシニアです!!(割引します!!)")
# 実行結果: ageの値が、18未満か、65以上のときに、
#         "あなたは未成年かシニアです!!(割引します!!)"が表示される
```

### andを使ってみる

"and"を使った記述例は次の通りです。

```python:main.py
age = 25
if age >= 18 and age <= 65:
    print("あなたは成人です!!(残念、割引はありません!!)")
# 実行結果: ageの値が、18以上で65以下のときに、
#         "あなたは成人です!!(残念、割引はありません!!)"が表示される
```

"not, or, and"を理解することで、より細かい分岐処理を実装できるようになります。
