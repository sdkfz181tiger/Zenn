# Python3で条件分岐を使ってみよう

今回のテーマは"条件分岐"です。
条件分岐を使うことで、複雑な処理を実装することができます。

## 真偽値と比較演算子

条件分岐の話をする前に、"真偽値"と"比較演算子"について解説をします。

### 真偽値とは

真偽値とは、"真"か"偽"のいずれか2種類のデータだけを表すデータ型のことです。

"真"は真実(しんじつ)の意味を表す"True"(トゥルー)で表現されます。
反対に、"偽"は偽物(にせもの)の意味を表す"False"(フォルス)で表現されます。

マルバツクイズの、マル(あたり)とバツ(はずれ)の関係と考えるとイメージしやすいです。
条件分岐では、この真偽値を使って判定を行います。

### 比較演算子とは

比較演算子とは、その名の通りデータ同士を"比較"する記号のことです。
比較した結果は、"True"または"False"の真偽値として返されます。
条件分岐では、この判定結果に従い処理の実行可否を決定します。

主な比較演算子は次のとおりです。

| 比較演算子 | 記述例 | 意味 |
| ---- | ---- | ---- |
| == | a == b | a, bの値が同じであればTrue, それ以外はFalse |
| != | a != b | a, bの値が違う場合はTrue, それ以外はFalse |
| < | a < b | aがb未満であればTrue, それ以外はFalse |
| <= | a <= b | aがb以下であればTrue, それ以外はFalse |
| > | a > b | aがbを超えていればTrue, それ以外はFalse |
| >= | a >= b | aがb以上であればTrue, それ以外はFalse |

## 条件分岐を使ってみる

条件分岐の具体的な記述方法について見ていきます。

### if文を使ってみる

条件分岐では、"if"キーワードを使って次のように記述します。
インデント(字下げ)によって、「条件が"True"のときに実行される処理のかたまり」を表します。

```python:main.py
if 条件:
    条件が"True"だった時に実行される
```

比較演算子を使った簡単な例をみてみます。

次の例では、条件が"True"と判定され、print()文が実行されます。

```python:main.py
age = 18
if age >= 18:
    print("あなたは大人です!!")
# 実行結果: あなたは大人です!!
```

しかし、ageの値を17にして実行した場合、
条件が"False"となり、print()文は実行されません。

```python:main.py
age = 17
if age >= 18:
    print("あなたは大人です!!")
# 実行結果: 条件が"False"のため、何も表示されません
```
