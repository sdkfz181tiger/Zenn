# Pythonで複数のデータを使ってみよう_後編

今回は、リストによく似た、"タプル"と"セット"、
そして"ディクショナリ"について学びます。

## タプル

タプルは、リストと非常によく似ています。
"("と、")"の間に、","区切りで値を並べて記述します。

リストと違い、タプルに含まれるデータは変更出来ない(immutable)という点だけ押さえておけばOKです。
よって、次の例ではエラーが起きてしまいます。

```python:main.py
member = ("まる子", "たま", "丸尾")
member[0] = "ばつ子" # これはエラー
member[1] = "ぽち" # これもエラー
member[2] = "罰男" # もちろんエラー
```

"絶対に変更されたく無いデータ"を扱うとき、タプルは有効な選択肢となります。

## セット

セットも、リストと非常によく似ています。
"{"と、"}"の間に、","区切りで値を並べて記述します。

セットは、"重複データを排除"するという特徴があります。
セットは順番を持たない(順不同)ので、member[0]のようなアクセスはできません。

```python:main.py
member = {"まる子", "野口", "たま", "野口", "丸尾", "野口"}
print(member)
# {'野口', 'まる子', '丸尾', 'たま'}
```

重複して存在していた"野口"が1つにまとめられています。

## ディクショナリ

ディクショナリは、その名の通り"辞書"という意味です。
データを"キー"と"値"をセットで管理します。
"値"にアクセスする時は、対応する"キー"を指定します。

```python:main.py
{ キー: 値 }
```

ディクショナリの使用例は次の通りです。

```python:main.py
member = {"maruchan": "まるこ", "tama": "たま", "maruo": "丸尾"}
print(member)
# {'maruchan': 'まるこ', 'tama': 'たま', 'maruo': '丸尾'}

# キーを指定して値にアクセスする
print(member["maruchan"]) # まるこ
print(member["tama"])     # たま
print(member["maruo"])    # 丸尾

# キーを指定して値を入れ替える
member["tama"] = "ぽち"
member["maruo"] = "罰男"
print(member)
# {'maruchan': 'まるこ', 'tama': 'ぽち', 'maruo': '罰男'}
```

### ディクショナリに用意された様々なメソッド

ディクショナリにも、いくつか重要な"メソッド"が存在します。
この3種類だけ押さえればOKです。

#### keys(キーの一覧を返す)

"keys()"メソッドを使う事で、"キー"の一覧を取り出すことができます。

```python:main.py
member = {"maruchan": "まるこ", "tama": "たま", "maruo": "丸尾"}
print(list(member.keys()))
# ['maruchan', 'tama', 'maruo']
```

#### values(値の一覧を返す)

"values()"メソッドを使う事で、"値"の一覧を取り出すことができます。

```python:main.py
member = {"maruchan": "まるこ", "tama": "たま", "maruo": "丸尾"}
print(list(member.values()))
# ['まるこ', 'たま', '丸尾']
```

#### items(キー:値の一覧を返す)

"items()"メソッドを使う事で、(キー, 値)のタプルの一覧を取り出すことができます。

```python:main.py
member = {"maruchan": "まるこ", "tama": "たま", "maruo": "丸尾"}
print(list(member.items()))
# [('maruchan', 'まるこ'), ('tama', 'たま'), ('maruo', '丸尾')]
```
