# Pythonの開発環境を構築する

ここでは、Pythonの開発環境の構築手順を紹介します。
2章の"Google Colab"で学習を進める方は、この章はスキップし、3章に進んでくださいね。

## 1, Pythonのダウンロード&インストール
[公式サイト](https://www.python.org/downloads/)から"Python"のインストーラーをダウンロード&インストールします。
インストール時に要求される、"Add to PATH"には必ずチェックを入れてインストールしてください。

![](/images/118d527876a482/03_install_01.png)

## 2, Pythonを実行してみよう

インストールが完了したら、動作確認をしてみます。
ターミナル(Macの場合)やコマンドプロンプト(Winの場合)を立ち上げ、
Pythonのバージョン情報を表示させてみましょう。

### ターミナルを立ち上げる(Macの場合)

次の手順で"Terminal.app"をクリックしターミナルを立ち上げます。
Finder -> Applications -> Utilities -> Terminal.app

ターミナル画面で、次のコマンド($から後の文字列)を入力し、"Enter"キーを押します。
※Macの場合は、"python3"で実行します。

```command:コマンド
# Mac は python3
$ python3 -V
Python 3.13.2
```

バージョン情報が表示されれば確認成功です。

### コマンドプロンプトを立ち上げる(Winの場合)

次の手順で"コマンドプロンプト"を立ち上げます。
スタートメニュー -> 「すべてのアプリ」もしくは「Windowsシステムツール」-> コマンドプロンプト

コマンドプロンプトで、次のコマンド(C:\>から後の文字列)を入力し、"Enter"キーを押します。
※Winの場合は、"python3"ではなく、"python"で実行します。

```command:コマンド
# Win は python
C:\> python -V
Python 3.13.2
```

バージョン情報が表示されれば確認成功です。

### エラーが出たら...

この時点でよくあるエラーを紹介します。

"python: command not found"
このエラーが出た時は、インストール時に"PATH"が設定されていない可能性があります。
その場合は、もう一度インストーラを開いて「Add to PATH」のチェックを入れて再インストールしてください。

## 3, Pythonで挨拶してみよう

次は、プログラムを記述するファイルを用意した上で、
コマンドから実行させてみましょう。

### プログラムを格納するフォルダを作る

先ずははじめの一歩です。
あなたのPCのデスクトップに、
次の手順で空のフォルダを作ります。

Macの場合: Desktop -> 右クリック -> "新規フォルダ"
Winの場合: Desktop -> 右クリック -> "新規フォルダ"

新しいフォルダの名前は何でも良いのですが、
ここではわかりやすく"hello_python3"としておきます。

### プログラムを記述するファイルを作る

次に、"hello_python3"フォルダの中にテキストファイルを作ります。
新しいファイルの名前は、"main.py"にしておきます。
今後はこのファイルにコードを記述していきます。

### プログラムを書いてみる

"main.py"を、何かしらのテキストエディタ等で開き、
次のコードを入力します。
(コピー&ペーストでも良いですよ)

```python: main.py
print("Hello, Python!!")
```

記述されているコードが今回の主役、"print(...)"という命令です。
最も簡単な命令の一つで「文字列を表示するコード」です。
"..."の部分に、表示したい文字列を記述します。

### プログラムを実行してみる

"main.py"を保存(忘れずに!!)したら、
ターミナル/コマンドプロンプトを開き、次のコマンド(2つあります)を実行します。

"hello_python3"フォルダに移動するコマンド"cd"と、
Pythonを実行するコマンド"python3"を実行します。

```command:コマンド(Macの場合)
# フォルダに移動するコマンド(cd)
$ cd ~/Desktop/hello_python3
# Pythonを実行するコマンド(Mac は python3)
$ python3 main.py
Hello, Python!! <- 実行結果
```

```command:コマンド(Winの場合)
# フォルダに移動するコマンド(cd)
C:\> cd C:\Users\<あなたのユーザー名>\Desktop\hello_python3
# Pythonを実行するコマンド(Win は python)
C:\> python main.py
Hello, Python!! <- 実行結果
```

"Hello, Python!!"の文字が表示されていれば大成功です!!
最初の難関をクリアしました。(おめでとうございます!!)

### エラーが出たら...

この時点でよくあるエラーを紹介します。

"python: command not found"
このエラーが出た時は、インストール時に"PATH"が設定されていない可能性があります。
その場合は、もう一度インストーラを開いて「Add to PATH」のチェックを入れて再インストールしてください。

"No such file or directory"
このエラーが出た時は、フォルダ移動できてない可能性があります。
フォルダ名を再確認し、もう一度cdコマンドで移動しましょう。
