# Python3で複数のデータを使ってみよう

今回のテーマは"複数のデータ"です。
一度に沢山のデータをまとめて管理する事ができるようになります。

## リスト

"list"を使う事で、複数のデータをまとめて管理することができます。
"["と、"]"の間に、","区切りで値を並べて記述します。

```python:main.py
member = ["まる子", "たま", "まるお"]
print(member)# 全てのデータを確認する
# ['まる子', 'たま', 'まるお']
```

リストにある値の1つだけにアクセスする方法は次のとおりです。
先頭の要素は[0]、2番目の要素は[1]、3番目の要素は[2]という具合にアクセスします。

```python:main.py
member = ["まる子", "たま", "まるお"]
print(member[0])# まる子
print(member[1])# たま
print(member[2])# まるお
```

リストの各要素を入れ替える場合は、次のように記述します。

```python:main.py
member = ["まる子", "たま", "まるお"]
member[1] = "ハマジ"
member[2] = "ブー太郎"
print(member)
# ['まる子', 'ハマジ', 'ブー太郎']
```
