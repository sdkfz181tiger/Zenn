# Python3で複数のデータを使ってみよう

今回は、リストによく似た、"タプル"と"セット"、
そして"ディクショナリ"について学びます。

## タプル

タプルは、リストと非常によく似ています。
"("と、")"の間に、","区切りで値を並べて記述します。

リストと違い、タプルに含まれるデータは変更出来ない(immutable)という点だけ押さえておけばOKです。
次の例ではエラーが起きてしまいます。

```python:main.py
member = ("まる子", "たま", "丸尾")
member[0] = "ばつ子" # これはエラー
member[1] = "ぽち" # これもエラー
member[2] = "罰男" # もちろんエラー
```

"絶対に変更されたく無いデータ"を扱うとき、タプルは有効な選択肢となります。

## セット

セットも、リストと非常によく似ています。
"{"と、"}"の間に、","区切りで値を並べて記述します。

セットは、"重複データを排除"するという特徴があります。
更に、データ順は順不同となるので注意しましょう。

```python:main.py
member = {"まる子", "野口", "たま", "野口", "丸尾", "野口"}
print(member)
# {'野口', 'まる子', '丸尾', 'たま'}
```

重複して存在していた"野口"が1つにまとめられています。

## ディクショナリ

ディクショナリは、その名の通り"辞書"という意味です。
データを"キー"と"値"の一セットとして管理します。

```python:main.py
{ キー: 値 }
```
