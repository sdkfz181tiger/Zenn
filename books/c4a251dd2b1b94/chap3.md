# Python3で挨拶してみよう

今回からコーディングのスタートです。
コードを記述するファイルの準備と実行手順を学びます。

## プログラムを格納するフォルダを作る

先ずははじめの一歩です。
あなたのPCのデスクトップに、
次の手順で空のフォルダを作ります。

Macの場合: Desktop -> 右クリック -> "新規フォルダ"
Winの場合: Desktop -> 右クリック -> "新規フォルダ"

新しいフォルダの名前は何でも良いのですが、
ここではわかりやすく"hello_python3"としておきます。

## プログラムを記述するファイルを作る

次に、"hello_python3"フォルダの中にテキストファイルを作ります。
新しいファイルの名前は、"main.py"にしておきます。
今後はこのファイルにコードを記述していきます。

## プログラムを書いてみる

"main.py"を、何かしらのテキストエディタ等で開き、
次のコードを入力します。
(コピー&ペーストでも良いですよ)

```python: main.py
print("Hello, Python3!!")
```

記述されているコードが今回の主役、"print(...)"という命令です。
最も簡単な命令の一つで「文字列を表示するコード」です。
"..."の部分に、表示したい文字列を記述します。

## プログラムを実行してみる

"main.py"を保存(忘れずに!!)したら、
ターミナル/コマンドプロンプトを開き、次のコマンド(2つあります)を実行します。

"hello_python3"フォルダに移動するコマンドと、
Python3を実行するコマンドを実行します。

```command:コマンド(Macの場合)
# フォルダに移動するコマンド
$ cd ~/Desktop/hello_python3
# Python3を実行するコマンド
$ python3 main.py
Hello, Python3!! <- 実行結果
```

```command:コマンド(Winの場合)
# フォルダに移動するコマンド
C:\> cd C:\Users\<あなたのユーザー名>\Desktop\hello_python3
# Python3を実行するコマンド
C:\> python main.py
Hello, Python3!! <- 実行結果
```

"Hello, Python3!!"の文字が表示されていれば大成功です!!
最初の難関をクリアしました。(おめでとうございます!!)

## コメントについて

コードの中に、"#"を記述することで、
コードの中に"メモ"を残すことができます。

Python3では、"#"が現れた後ろの1行を"コメント"と認識し、
実行時に無視をしてくれます。

コメントは次の様に、"覚書き"として使う事が多いでしょう。
(次回、このコードを見たときに全てを思い出す事ができます!!)

```python:main.py
print("Hello, Python3!!")# 文字を表示する命令
```

複数行のコメントを残す時は、"""(トリプルクォート)を使います。

```python:main.py
"""
以下のコードは、
文字列を表示する、
とても重要なコードです!!
"""
print("Hello, Python3!!")
```

今後、あなたがコードを記述していく過程で、
気づいた事や思い出したい事等思いついた時は、
積極的にメモを残していきましょう。

## エラーが出たら...

この時点でよくあるエラーを紹介します。

"python: command not found"
このエラーが出た時は、インストール時に"PATH"が設定されていない可能性があります。
その場合は、もう一度インストーラを開いて「Add to PATH」のチェックを入れて再インストールしてください。

"No such file or directory"
このエラーが出た時は、フォルダ移動できてない可能性があります。
フォルダ名を再確認し、もう一度cdコマンドで移動しましょう。
