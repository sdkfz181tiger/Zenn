# Python3って何?

## どんな言語?

[Python3](https://www.python.org/downloads/)は、シンプルで読みやすいことを大切に作られたプログラミング言語です。
専門用語が少なくコードもスッキリしているので、プログラミング初学者向けの言語としてとても人気があります。

ちなみに "Python"という名前は"ヘビ"ではなく、
イギリスのコメディ番組"[Monty Python's Flying Circus](https://ja.wikipedia.org/wiki/%E3%83%A2%E3%83%B3%E3%83%86%E3%82%A3%E3%83%BB%E3%83%91%E3%82%A4%E3%82%BD%E3%83%B3)"から取られています。
ちょっとユニークで覚えやすいですね。

![](/images/118d527876a482/01_01.png)

## どんなところで使われてる?

Python3はいろいろな場面で使われていて、
2Dゲーム開発、業務の自動化、Web開発、IoT、データ分析など、
活躍する分野がとても広いのも特徴です。
特に最近では、AI(機械学習)用のライブラリの豊富さが注目され、
データサイエンスの世界で定番言語になっています。

Python3の基本的な使い方をマスターした後は、
必要なライブラリを組み合わせて、様々な物を形にしていくことができる様になります。
(みなさんは何が作りたいですか?)

### 2Dゲーム開発

筆者はレトロゲームが大好きなので、
ここで2Dゲーム用のライブラリをいくつか紹介したいと思います。

#### Pyxel

"[Pyxel](https://github.com/kitao/pyxel)"は、レトロな8bit風ゲーム制作に特化した、かわいいゲームエンジンです。
"ファミコンっぽい雰囲気"が好きな人におすすめです。(筆者激推し)

![](/images/118d527876a482/01_02.png)

#### pygame

"[pygame](https://www.pygame.org/news)"は、歴史が長い2Dゲームライブラリです。
チュートリアルやサンプルが豊富で、
シンプルな2Dゲームを作りたい人に向いています。

![](/images/118d527876a482/01_03.png)

#### Python Arcade Library

"[Python Arcade Library](https://api.arcade.academy/en/stable/)"は、本格的な描画機能を持ちながら扱いやすい、人気が伸びているライブラリ。
教育向けにも広く使われています。

![](/images/118d527876a482/01_04.png)
