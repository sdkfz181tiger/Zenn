# Pythonで計算してみよう

今回のテーマは"計算"です。

## 四則演算

Pythonでは、四則演算を使って次のように計算を行います。
上から順番に、足し算、引き算、掛け算"\*"、割り算"\/"です。

```python:main.py
# 簡単な計算
print(1 + 2)# 結果: 3
print(4 - 3)# 結果: 1
print(5 * 6)# 結果: 30
print(8 / 4)# 結果: 2.0 (割り算の結果は小数になります)
```

掛け算と割り算は、足し算と引き算より先に実行されます。
"("と")"を使う事で、計算の順番を優先させる事ができます。

```python:main.py
# 少し複雑な計算
print(1 + 2 * 3)# 結果: 7
print((1 + 2) * 3)# 結果: 9
```

"\*"を2個続けて書く事で、べき乗を計算する事ができます。

```python:main.py
# べき乗
print(2**2)# 2の2乗 = 結果: 4
print(2**3)# 2の3乗 = 結果: 8
print(2**4)# 2の4乗 = 結果: 16
```

## コメントについて

コードの中に、"#"を記述することで、
"メモ"を残すことができます。

Pythonでは、"#"が現れた後ろの1行を"コメント"と認識し、
実行時に無視をしてくれます。

コメントは次の様に、"覚書き"として使う事が多いでしょう。
(次回、このコードを見たときに全てを思い出す事ができます!!)

```python:main.py
print("Hello, Python!!")# 文字を表示する命令
```

複数行のコメントを残す時は、"""(トリプルクォート)を使います。

```python:main.py
"""
以下のコードは、
文字列を表示する、
とても重要なコードです!!
"""
print("Hello, Python!!")
```

気づいた事等、積極的にコメントを残していきましょう。
