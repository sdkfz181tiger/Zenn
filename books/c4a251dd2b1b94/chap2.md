# Python3を"Google Colab"で実行する

ここでは、気軽にPython3を試せるオンラインサービス、"Google Colab"を紹介します。
このサービスを利用することで、開発環境の準備が不要になります。
(Python3を気軽に触れてみたい方はコチラがオススメです)

## 1, Google Coloabにアクセスする

"Google Coloab"にアクセスします。
(Googleのアカウントが必要になります)

左上メニューから、次の順番でクリックし、"新しいノートブック"を立ち上げます。
ファイル -> ドライブの新しいノートブック

![](/images/118d527876a482/02_colab_01.png)

## 2, コードを入力して実行する

新しいノートブックが立ち上がったら、準備完了です。

赤枠の部分に、次のコードを入力し、直ぐ左にあるRunボタンで実行します。

```python3: Google Colab
print("Hello, Python3!!")
```

![](/images/118d527876a482/02_colab_02.png)

直ぐ下の行に"Hello, Python3!!"と表示されれば成功です。
とても簡単ですね!!

Python3を気軽に試したい程度であれば、"Google Colab"で十分です。

本格的な開発をしたい場合は、"第3章: 実行してみよう(インストール編)"を参考してチャレンジしてみてください。
