# ターミナルで遊んでみよう
ターミナルには様々なコマンドが存在しますが、
今回は簡単に使えるジョークコマンドを5つ紹介します。
ターミナルを楽しむきっかけになるはずです!!

![](/images/articles/0a3fc896ebdb07/2882affca9fc-20241004.png)
*cowsay*

# 1,cowsay
その名の通り、牛が好きなセリフを言ってくれます。
ジョークコマンドの王様と言っても過言ではないかも!?

1-1. コマンドのインストール
```shell: command
# Linuxはこちら
sudo apt install cowsay
# Macはこちら
brew install cowsay
```

1-2. コマンドの実行
```shell: command
cowsay nice to meet you
```

![](/images/articles/0a3fc896ebdb07/2882affca9fc-20241004.png)
*cowsay*
