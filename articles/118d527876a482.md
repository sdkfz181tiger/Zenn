# はじめに
Python3エンジニア認定[基礎試験](https://www.pythonic-exam.com/exam/basic)の受験に必要な文法を中心にまとめていきます。
受験直前の確認用に使っていただけると幸いです。

[実践試験の記事はコチラ]()

## Python3エンジニア認定試験とは

[Python3エンジニア認定試験](https://www.pythonic-exam.com/)には、
主にその文法等が問われる"知識系"とデータ分析力が問われる"データ分析系"の2種類存在します。

| 大別 | 試験名 | 内容 |
| ---- | ---- | ---- |
| 知識系 | [基礎試験](https://www.pythonic-exam.com/exam/basic) | Python3の基礎知識や文法知識が問われる |
|  | [実践試験](https://www.pythonic-exam.com/exam/jissen) | 基礎試験の上位試験 |
| データ分析系 | [データ分析試験](https://www.pythonic-exam.com/exam/analyist) | Python3を使ったデータ分析に関する基礎知識が問われる |
|  | [データ分析実践試験](https://www.pythonic-exam.com/exam/cpda) | データ分析試験の上位試験 |

Python3が初めての方は、
"知識系"の[基礎試験](https://www.pythonic-exam.com/exam/basic) -> "データ分析系"の[データ分析試験](https://www.pythonic-exam.com/exam/analyist)を狙う流れがシンプルで分かり易いと思います。

# Python3の特徴

"Python3"はコンパイルが不要なインタープリタ言語であり、その構文も非常にコンパクトに設計されている事から初学者向けの言語として広く使われています。
そして、AI分野である機械学習向けのライブラリが豊富な事からデータサイエンティスト向けの言語として非常に人気があります。

ちなみに、"Python"という洒落た名前はイギリスのコメディ番組"[Monty Python's Flying Circus](https://ja.wikipedia.org/wiki/%E3%83%A2%E3%83%B3%E3%83%86%E3%82%A3%E3%83%BB%E3%83%91%E3%82%A4%E3%82%BD%E3%83%B3)"から来ています。(ヘビではありません...)

# Python3のインストールと実行方法

Python3の実行環境は[公式サイト](https://www.python.org/downloads/)からインストーラーをダウンロードし、インストールするだけで直ぐに使える様になります。

![](/images/118d527876a482/01.png)

プログラムを記述した"main.py"ファイルを用意した後、次のコマンドで実行します。
(必要であればPATHを通します)

```shell: output
$ python3 main.py
```
