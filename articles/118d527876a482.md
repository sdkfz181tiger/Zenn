# はじめに
Python3エンジニア認定[基礎試験](https://www.pythonic-exam.com/exam/basic)の受験に必要な文法を中心にまとめていきます。
受験直前の確認用に使っていただけると幸いです。

## Python3エンジニア認定試験とは

[Python3エンジニア認定試験](https://www.pythonic-exam.com/)には、
主にその文法等が問われる"知識系"とデータ分析力が問われる"データ分析系"の2種類存在します。

| 大別 | 試験名 | 内容 |
| ---- | ---- | ---- |
| 知識系 | [基礎試験](https://www.pythonic-exam.com/exam/basic) | Python3の基礎知識や文法知識が問われる |
|  | [実践試験](https://www.pythonic-exam.com/exam/jissen) | 基礎試験の上位試験 |
| データ分析系 | [データ分析試験](https://www.pythonic-exam.com/exam/analyist) | Python3を使ったデータ分析に関する基礎知識が問われる |
|  | [データ分析実践試験](https://www.pythonic-exam.com/exam/cpda) | データ分析試験の上位試験 |

Python3が初めての方は、
"知識系"の[基礎試験](https://www.pythonic-exam.com/exam/basic) -> "データ分析系"の[データ分析試験](https://www.pythonic-exam.com/exam/analyist)を狙う流れがシンプルで分かり易いと思います。

# Python3の特徴

"Python3"はコンパイルが不要なインタープリタ言語であり、その構文も非常にコンパクトに設計されている事から初学者向けの言語として広く使われています。
そして、AI分野である機械学習向けのライブラリが豊富な事からデータサイエンティスト向けの言語として非常に人気があります。

ちなみに、"Python"という洒落た名前はイギリスのコメディ番組"[Monty Python's Flying Circus](https://ja.wikipedia.org/wiki/%E3%83%A2%E3%83%B3%E3%83%86%E3%82%A3%E3%83%BB%E3%83%91%E3%82%A4%E3%82%BD%E3%83%B3)"から来ています。(ヘビではありません...)

# Python3のインストールと実行方法

Python3の実行環境は[公式サイト](https://www.python.org/downloads/)からインストーラーをダウンロードし、インストールするだけで直ぐに使える様になります。

![](/images/118d527876a482/01.png)

プログラムを記述した"main.py"ファイルを用意した後、次のコマンドで実行します。
(必要であればPATHを通します)

```shell: コマンド
$ python3 main.py
```

# 01: テキストの操作

文字列は''と、""のいずれでも使う事が可能です。

```python: main.py
print('THE WORLD!!')# THE WORLD!!
print("THE WORLD!!")# THE WORLD!!
```

"\\"(バックスラッシュ)を使う事で特殊文字をエスケープをする事が可能です。

```python: main.py
print('doesn\'t')# doesn't (\使う場合)
print("doesn't")# doesn't (\使わない場合)
print('"TO BE CONTINUED...", they said.')# "TO BE CONTINUED...", they said.
print("\"TO BE CONTINUED...\", they said.")# "TO BE CONTINUED...", they said.
```

複数行にまたがる場合は'''か、"""(トリプルクォート)を使います。
行の最初と末尾で改行を除きたいときは"\\"(バックスラッシュ)を使います。

```python: main.py
print("""\
きさま!
見ているなっ!\
""")
# きさま!
# 見ているなっ!
```

"\*"をつかうことで繰り返し、"\+"を使う事で連結する事が可能です。

```python: main.py
print("無駄" * 5 + "ァ!!")# 無駄無駄無駄無駄無駄ァ!!
```

""(クォーテーション)で囲まれた文字列は自動的に連結されます。
(長い文字を扱う時に有効です)

```python: main.py
print("花京院くん、"
"恐れることはないんだよ。"
"友だちになろう。")
# 花京院くん、恐れることはないんだよ。友だちになろう。
```

文字列を変数に格納して扱う事ができます

```python: main.py
message = "ドアぐらい開けて出ていけ・・・・・"
print(message)# ドアぐらい開けて出ていけ・・・・・
```

インデックス指定する事で1文字だけ取り出す事ができます。

```python: main.py
message = "この世界の空間から姿をまったく消すスタンドよ"
print(message)# この世界の空間から姿をまったく消すスタンドよ
print(message[2])# 世
print(message[3])# 界
```

文字列のインデックス位置は次の例を参考にします。

```python: インデックス位置
#  +---+---+---+---+---+
#  | W | o | r | l | d |
#  +---+---+---+---+---+
#  0   1   2   3   4
# -5  -4  -3  -2  -1
```

文字列自体は不変体(immutable)なので、インデックス指定で置き換える事はできません。

```python: main.py
message = "この世界の空間から姿をまったく消すスタンドよ"
message[3] = "間" # ここでエラー
```

スライスを使う事で文字列の一部分を切り出す事が可能です。

```python: main.py
message = "そうかそうかポルナレフ、階段を降りたな。このDIOの仲間になりたいという訳だな。"
print(message[6])# ポ
print(message[6:11])# ポルナレフ
print(message[22])# D
print(message[22:25])# DIO
print(message[:11])# そうかそうかポルナレフ
print(message[26:])# 仲間になりたいという訳だな。
print(message[-4:])# 訳だな。
```

変数に格納した文字列は"\+"を使う事で連結する事が可能です。

```python: main.py
emo1 = "動揺"
emo2 = "恐怖"
print("「" + emo1 + "する」それはつまり「" + emo2 + "している」ということでは無いのかね。")
# 「動揺する」それはつまり「恐怖している」ということでは無いのかね。
```

文字列の長さは、len関数で取得が可能です。

```python: main.py
message = "「長所」と「短所」は表裏一体・・・ままならぬものよ・・・"
print("文字数は" + str(len(message)) + "です")# 文字数は28です
```

# 02: リストの操作

複数のデータをまとめて扱うときはリストを使います。
文字列と同様にしてスライスでその一部のコピーを取得する事が可能です。

```python: main.py
squares = [1, 4, 9, 16, 25]
print(squares)# [1, 4, 9, 16, 25]
print(squares[0])# 25
print(squares[-3:])# [9, 16, 25]
```

リスト同士を"\+"で連結する事ができます。

```python: main.py
squares = [1, 4, 9, 16, 25]
squares = squares + [36, 49, 64, 81, 100]
print(squares) # [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]
```

リストは可変体(mutable)なので、内容を直接入れ替える事が可能です。

```python: main.py
cubes = [1, 8, 27, 65, 125]
cubes[3] = 64
print(cubes)# [1, 8, 27, 64, 125]
```

リストに値を追加するときは、"append"を使います。

```python: main.py
cubes.append(6 ** 3)# 6の3乗
cubes.append(7 ** 3)# 7の3乗
print(cubes)# [1, 8, 27, 64, 125, 216, 343]
```

スライスでリストの一部を置き換える事が可能です。

```python: main.py
letters = ["a", "b", "c", "d", "e", "f", "g"]
print(letters)# ['a', 'b', 'c', 'd', 'e', 'f', 'g']

# 一部の範囲を置き換える
letters[2:5] = ["C", "D", "E"]
print(letters)# ['a', 'b', 'C', 'D', 'E', 'f', 'g']

# 一部の範囲を削除する
letters[2:5] = []
print(letters)# ['a', 'b', 'f', 'g']

# 全てを削除する
letters[:] = []
print(letters)# []
```
