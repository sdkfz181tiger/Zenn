## Python3エンジニア認定試験とは

[Python3エンジニア認定試験](https://www.pythonic-exam.com/)には、
主にその文法等が問われる"知識系"とデータ分析力が問われる"データ分析系"の2種類存在します。

| 大別 | 試験名 | 内容 |
| ---- | ---- | ---- |
| 知識系 | [基礎試験](https://www.pythonic-exam.com/exam/basic) | Python3の基礎知識や文法知識が問われる |
|  | [実践試験](https://www.pythonic-exam.com/exam/jissen) | 基礎試験の上位試験 |
| データ分析系 | [データ分析試験](https://www.pythonic-exam.com/exam/analyist) | Python3を使ったデータ分析に関する基礎知識が問われる |
|  | [データ分析実践試験](https://www.pythonic-exam.com/exam/cpda) | データ分析試験の上位試験 |

Python3が初めての方は、
"知識系"の[基礎試験](https://www.pythonic-exam.com/exam/basic) -> "データ分析系"の[データ分析試験](https://www.pythonic-exam.com/exam/analyist)を狙う流れがシンプルで分かり易いと思います。
